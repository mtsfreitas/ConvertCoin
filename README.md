# CurrencyConverter

App iOS de conversão de moedas/câmbio, desenvolvido usando o Swift 5. 

- Consumo de API Rest pública (https://docs.awesomeapi.com.br/api-de-moedas) utilizando URLSession 
- Gerenciador de dependências: CocoaPods
- Arquitetura: MVVM
